ZeldaJS

The Legend of Zelda, recreated in TypeScript.

Download this project as a .zip file Download this project as a tar.gz file

What is it?

This is a recreation of The Legend of Zelda for the NES in TypeScript. It aims to be as authentic as possible, warts and all. The feature set currently includes:

  • Smooth scrolling between screens
  • Use of sword and bombs
  • Enemies that can die
  • Entering new maps (caves and levels)

Demo

Use the arrow keys to move around, Z to use the sword, and X to use a bomb.
